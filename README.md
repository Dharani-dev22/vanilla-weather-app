# 🌤️ Vanilla JavaScript Weather Dashboard

> **Live Demo:** [https://vanilla-weather-app-three.vercel.app/](https://vanilla-weather-app-three.vercel.app/)

A lightweight, fully responsive weather application built entirely from scratch without the use of frontend frameworks or external CSS libraries. This project demonstrates core web development fundamentals, including direct DOM manipulation, asynchronous network requests, and modern CSS architecture.

## 🚀 Technical Highlights

This application intentionally avoids tools like React or Tailwind to showcase a deep understanding of browser-native technologies:

* **Asynchronous Data Fetching:** Utilizes modern ES6 `async/await` syntax and the native `fetch` API to retrieve live, real-world meteorological data from the open-source `wttr.in` network.
* **Robust Error Handling:** Implemented strict `try/catch` blocks to gracefully handle network failures or invalid user inputs without crashing the application.
* **Modern CSS Architecture:** Built entirely with Vanilla CSS utilizing **CSS Variables** for centralized theme management, **Flexbox** for responsive component alignment, and **CSS Transitions** for smooth, state-based UI animations.
* **Direct DOM Manipulation:** Bypasses virtual DOMs to directly interface with browser nodes, ensuring a deep understanding of event listeners and element updates.

## 🛠️ Tech Stack

* **Structure:** Semantic HTML5
* **Styling:** Vanilla CSS3
* **Logic:** Vanilla JavaScript (ES6+)
* **API:** `wttr.in` (Keyless Weather API)

## 💻 Run Locally

1. Clone the repository:
    git clone https://github.com/Dharani-dev22/vanilla-weather-app.git

2. Navigate to the directory:
    cd vanilla-weather-app

3. Open `index.html` in your preferred web browser.
